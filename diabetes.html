Diabetes Prediction

A machine learning model that predicts the likelihood of diabetes based on various health indicators.

Project Overview

This project predicts the probability of a patient developing diabetes based on health features like age, BMI, and blood pressure.

Technologies Used

  • Python (Pandas, NumPy)
  • Scikit-Learn
  • Logistic Regression
  • Matplotlib/Seaborn

Key Features

  • Data preprocessing and feature scaling
  • Logistic regression for prediction
  • Model evaluation with confusion matrix and ROC curve

Results

The model showed promising results with an accuracy of 85%. It identified key risk factors such as high BMI and age.
