Iris Flower Classification

A simple machine learning model that classifies iris flowers into three species based on their sepal and petal measurements.

Project Overview

This project uses the classic Iris dataset to build a classification model. I implemented and compared multiple algorithms to predict the species of iris flowers (Setosa, Versicolor, Virginica) based on their sepal and petal length and width.

Technologies Used

  • Python (Pandas, NumPy)
  • Scikit-Learn
  • Logistic Regression, K-Nearest Neighbors, Decision Trees
  • Matplotlib & Seaborn

Key Features

  • Exploratory data analysis (EDA)
  • Data visualization for class separation
  • Model training, testing, and evaluation
  • Comparison of model performance

Results

The best-performing model achieved over 95% accuracy. Visualizations confirmed clear separation between species, especially for Setosa.
