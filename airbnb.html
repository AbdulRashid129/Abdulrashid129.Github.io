Airbnb Data Analysis

Exploratory data analysis of Airbnb listings to understand pricing, availability, and neighborhood trends.

Project Overview

This project analyzes Airbnb listing data to uncover insights about pricing patterns, popular neighborhoods, and availability trends. The goal was to support data-driven decisions for hosts and travelers alike.

Technologies Used

  • Python (Pandas, NumPy)
  • Matplotlib & Seaborn
  • Jupyter Notebook
  • GeoPandas (optional, for maps)

Key Features

  • Price vs. location analysis
  • Listing availability trends
  • Top neighborhoods by number of listings and reviews
  • Distribution of room types and host response rates

Results

The analysis highlighted that central locations had higher prices but more reviews. Entire homes were priced significantly higher than shared rooms, and availability varied seasonally across neighborhoods.
