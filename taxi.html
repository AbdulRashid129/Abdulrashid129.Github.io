Taxi Trip Data Analysis

Analyzing NYC taxi trip data to uncover insights on demand patterns, trip duration, and fare structure using Python and data visualization tools.

Project Overview

This project explores a large dataset of New York City taxi trips to analyze travel patterns, trip lengths, and fare metrics. Using pandas and visualization libraries, I extracted key insights about city mobility and passenger behavior.

Technologies Used

  • Python (Pandas, NumPy)
  • Matplotlib & Seaborn
  • Jupyter Notebook
  • Geospatial data (lat-long coordinates)

Key Features

  • Exploratory data analysis (EDA)
  • Heatmaps of pickup and drop-off locations
  • Trip duration and fare distribution analysis
  • Hourly and weekday demand trends

Results

The analysis revealed peak taxi usage during rush hours and weekends, with Midtown Manhattan as the busiest zone. Visualizations helped communicate demand hotspots and fare trends effectively.
