Customer Segmentation

Unsupervised machine learning to segment customers based on spending behavior and demographics.

Project Overview

This project applies unsupervised learning (K-Means Clustering) to segment customers into groups based on purchasing patterns and personal attributes. It helps businesses identify key customer types for targeted marketing.

Technologies Used

  • Python (Pandas, NumPy)
  • Scikit-learn
  • K-Means Clustering
  • Matplotlib & Seaborn

Key Features

  • Data preprocessing and normalization
  • Elbow method to determine optimal clusters
  • 2D and 3D visualizations of customer segments
  • Insights for personalized marketing strategies

Results

The clustering algorithm identified distinct customer groups such as high spenders, budget-conscious shoppers, and moderate users. These insights can support customer retention and upselling strategies.
