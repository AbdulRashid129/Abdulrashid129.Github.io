Titanic Survival Model

A predictive model that uses logistic regression to predict the survival of passengers on the Titanic.

Project Overview

In this project, I used logistic regression to predict the likelihood of survival for passengers on the Titanic based on various features like age, gender, and class.

Technologies Used

  • Python (Pandas, NumPy)
  • Scikit-Learn
  • Logistic Regression
  • Matplotlib/Seaborn

Key Features

  • Data cleaning and feature engineering
  • Logistic regression for prediction
  • Model evaluation with accuracy, confusion matrix, and ROC curve

Results

The model achieved 78% accuracy on the test set. The results showed significant importance of the 'Sex' and 'Pclass' features in predicting survival.
